# Data manipulation

# Add data

# Single piece

When adding data, if the number or order of values is different from the number or order of fields defined in the data table, you must ** specify fields **. The specified fields can be all or some of the fields in the table

INSERT [INTO] Table name (Field name 1, field name 2,...) {VALUES|VALUE}
(Value 1, value 2,...) ;
The keyword INTO can be omitted, and the effect after omission is the same.
Field name Indicates the name of the field to which data is to be added. The order of field names must correspond to the order of values. Multiple field names are separated by commas (,).
VALUES and VALUE represent VALUES. You can choose either one. Values is usually used.
Value Indicates data corresponding to a field. Multiple values are separated by commas (,).

# Multiple

INSERT INTO table name [(Field name 1, field name 2,...)]  {VALUES|VALUE}
(The value of the first record 1, the value of the first record 2,...) ,
(The value of the second record 1, the value of the second record 2,...) ,
...
(Value 1 for the NTH record, value 2 for the NTH record,...) ;

If no field name is specified, the order of the values is the same as the order of the fields in the data table.
If a field name is specified, the order of the values is the same as the specified field name order.

# Update data

# part

Updating partial data refers to updating one or more data in a data table according to the WHERE clause that specifies the data update conditions.

UPDATE table name SET field name 1= value 1[, field name 2= value 2,...] WHERE conditional expression;
The -- SET clause specifies the name of the field in the table to be updated and the corresponding value.
The -- WHERE clause specifies the records in the table to be updated. WHERE is followed by a specified condition. Only records that meet the specified condition will be updated.

The field name in the SET clause is the name of the field to be updated, and the value is the value after the field is updated. If you want to update the original field value, you can use the add (+), subtract (-), multiply (*), divide (/) operators.
For example, if the value of age is 18, "age+1" means adding 1 to the original, resulting in 19.

# All of them

When you use the UPDATE statement to update data, if the WHERE clause is not added, all the specified fields of all the data in the table are updated.

UPDATE table name SET field name 1= value 1[, field name 2= value 2,...] ;
The -- SET clause specifies the name and corresponding value of the field to be modified in the table. The field name is the name of the field to be modified, and the value is the modified value of the corresponding field name.

# Delete data

# part

The DELETE statement can delete some data, and the WHERE clause can specify the conditions for deleting data.

DELETE FROM table name WHERE conditional expression;
The table name is the name of the data table to be deleted
The WHERE clause is used to set the deletion conditions. Data that meets the conditions will be deleted

# All of them

Use the DELETE statement to delete all the data in the data table, simply omit the WHERE clause when using it

DELETE FROM table name;

# Use TRUNCATE statement to delete all data in the data table

TRUNCATE [TABLE] Table name;

-- If the field value is set to the AUTO_INCREMENT attribute:
When TRUNCATE is used to delete data and TRUNCATE is used to add data again, the value of this field increases from 1.
- DELETE data using the DELETE statement, the field value does not increase from 1, but keeps the original automatic growth value.
